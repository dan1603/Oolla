# Oolla

This is a test application, which displays the list of GitHub user repositories.

#### Architecuture

- Pattern: MVVM
- Language: Kotlin

#### Components
- Dagger 2
- Room
- PagingLibrary
- LiveData
- DataBinding
- AndroidJetpack
